Hello to our wonderful API

This page will teach everything about the use of our API

The API allows you to query the weather for a given location for the next 10 days (and night).

The main part of the url is http://193.191.187.14:11008/WeatherApp-Rest/API/weather/ after which you can use {country}-{city}. The country has to be specified by its abbreviation like "BE". The city can ge given by its english name in some cases, or in its native language. An example of this would be http://193.191.187.14:11008/WeatherApp-Rest/API/weather/BE-Leuven

So in short all URL's are in the form http://193.191.187.14:11008/WeatherApp-Rest/API/weather/{country}-{city}

Cities that are conformed to be queryable are: BE-Leuven, BE-Gent, BE-Antwerpen, BE-Vilvoorde, BE-Brussels, BE-Liege. Around 200 000 cities are supported
